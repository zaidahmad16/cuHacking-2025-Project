# Assets Directory

This directory should contain the following files for the Expo application:

- `icon.png` - App icon (1024x1024 px recommended)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (foreground, 1024x1024 px)
- `favicon.png` - Web favicon (48x48 px or larger)

These files are referenced in `app.json` but are not included in the repository.
You'll need to create or add your own icon and splash screen assets.

## Quick Setup

If you don't have custom assets yet, you can:
1. Use Expo's default assets (they'll be generated automatically on first run)
2. Create placeholder images with your preferred design tool
3. Use online icon generators to create app icons

For more information, see: https://docs.expo.dev/develop/user-interface/splash-screen-and-app-icon/
